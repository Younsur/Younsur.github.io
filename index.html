我的第一篇博客

今天开始博客之旅，没想好写什么，就回顾一下怎么搭建博客吧​

零、前言

程序员计数从0开始，很合理。

本人目前是个大三计算机专业的菜鸟，尽管课内已经快实现一周五休的休闲生活，却还得面临考研、实习的诸多压力，又闲又累。在无所事事快一个月后，本着找点事做的想法，又考虑到找实习时可以加分，心血来潮想要搭建一个个人博客。

在上网搜索教程后，决定基于Hexo搭建个人博客。选择的原因再简单不过，有教程，易上手，经过一番捣鼓，也是成功搭建。因此，为了纪念搭建成功，也是记录下搭建的过程，写了这篇博客QAQ。

我是在Windows下搭建的，所以以下也是在Windows系统上搭建的方法。其他系统就等我深入学习后再更新吧。

博客新手，写的不好，还请海涵

不定期更新

一、安装git

把本地文件传到github上的强大工具，官网下载即可

下载完成后，右键—>git bash，即可在当前文件夹打开命令行敲命令了

二、安装nodejs

Hexo是基于node.js编写的，所以我们还需要安装下nodejs

windows直接去官网下载就好，Node.js，安装一路Next即可

安装后可以在命令行输入node -v和npm -v命令，如果出现版本号，就说明安装成功

三、安装Hexo

首先我们可以先建好一个新文件夹，用来存放博客和相关文件，文件夹的位置没有要求，能找到就行

比如我就放在C:\Users\10605\MyBlog

创建好后，右键文件夹，git bash here，打开命令行控制台

输入npm install -g hexo-cli安装Hexo

安装完成后，可以输入hexo -v验证是否安装成功

接下来，初始化hexo

再输入npm install安装必备组件

完成后，文件夹目录下应该有：

  • node_modules 依赖包
  • public 存放生成的页面
  • scaffolds 生成文章的一些模板
  • source 用来存放文章
  • themes 主题
  • _config.yml 博客的配置文件

此时本地的网站配置就弄好了，输入hexo g生成静态网页，输入hexo s打开本地服务器查看
